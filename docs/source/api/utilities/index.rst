Utilities
=========

This section documents the utility modules included in the project. These modules are organized into the following categories:

- :doc:`Controller Utilities <controller>`: Utilities for creating and simulating Data-Driven MPC controllers.
- :doc:`Model Utilities <models>`: Classes for modeling Linear Time-Invariant (LTI) and nonlinear systems.
- :doc:`Visualization Utilities <visualization>`: Utilities for generating static and animated plots of control input-output data.
- :doc:`Common Utilities <common_utilities>`: Shared utility functions used across modules (e.g., Hankel matrices, estate estimation, YAML parameter loading).
